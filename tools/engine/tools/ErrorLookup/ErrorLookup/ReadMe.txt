========================================================================
    ErrorLookup - 魔幻修仙客户端错误码解释器
========================================================================
V 1.0:

1. 魔幻修仙错误码分3种，分别如下所述：
	A. Windows API错误码，通过EC::GetErrorDescription()获取解释信息。
	B. PFS API错误码，通过本程序实现的PFS::GetErrorDescription()获取解释信息。
	C. PP API错误码，通过本程序实现的PP::GetErrorDescription()获取解释信息。
	
2. 待完善部分：
	A. 支持0xE0000000类型的16进制错误码解析。
	B. 支持动态加载用户选择的模块，类似于Visual Studio 2005提供的Error Lookup。
	C. 帮助功能。
