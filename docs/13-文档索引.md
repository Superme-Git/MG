## 阅读路线（按目标结构）

- 路线A：Windows快速上手（15 分钟）
  - 01 → 06 → 运行（按 06 的"方法 A"）

- 路线B：Windows完整编译（30–45 分钟）
  - 05 → 07 → 06（方法 A 或 B） → 验证与运行

- 路线C：Android编译（45-60 分钟）
  - 14 → android/03_环境配置指南.md → 编译与打包

- 路线D：架构与技术学习（1–2 小时）
  - 02 → 03 → 04 → 11 → 12（选读专题）

- 路线E：问题排查（即时）
  - 遇到编译错误 → 10（或 06 的故障排查章节）
